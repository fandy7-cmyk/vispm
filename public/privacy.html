Ringkasan: VISPM hanya mengumpulkan data yang diperlukan untuk verifikasi Standar Pelayanan Minimal (SPM) Puskesmas. Kami tidak menjual, menyewakan, atau membagikan data Anda kepada pihak ketiga untuk tujuan komersial.
1

Informasi yang Kami Kumpulkan

2

Bagaimana Kami Menggunakan Data

Data yang dikumpulkan digunakan semata-mata untuk keperluan:

  • Menjalankan sistem verifikasi dan pelaporan Indeks SPM Puskesmas
  • Autentikasi pengguna dan kontrol akses berdasarkan peran (Operator, Kapus, Pengelola Program, Admin)
  • Menyimpan riwayat verifikasi dan persetujuan untuk keperluan audit
  • Menghasilkan laporan resmi indeks SPM per periode
  • Mengunggah dan mengelola dokumen data dukung ke Google Drive milik pengguna
Kami tidak menggunakan data Anda untuk: iklan, analitik komersial, profiling, atau dibagikan ke pihak ketiga di luar keperluan operasional sistem ini.
3

Integrasi Google Drive

VISPM menggunakan Google OAuth 2.0 untuk mengakses Google Drive pengguna dalam rangka menyimpan dokumen data dukung indikator SPM.

Scope yang digunakan: https://www.googleapis.com/auth/drive.file
Akses ini hanya terbatas pada file yang dibuat oleh aplikasi VISPM. Kami tidak dapat membaca, mengubah, atau menghapus file lain di Google Drive Anda.

Rincian penggunaan integrasi Google Drive:

  • File data dukung diunggah ke folder yang Anda tentukan di Google Drive
  • Struktur folder mengikuti format: Nama-PKM / Tahun / Bulan / Indikator / No
  • Token akses disimpan di browser lokal (localStorage) dan tidak dikirim ke server kami
  • Token di-refresh secara otomatis menggunakan refresh token yang tersimpan lokal
  • Anda dapat mencabut akses kapan saja melalui pengaturan akun Google Anda

Penggunaan data dari Google API mematuhi Google API Services User Data Policy, termasuk ketentuan Limited Use.

4

Keamanan Data

Kami menerapkan langkah-langkah keamanan berikut untuk melindungi data Anda:

  • HTTPS: Seluruh komunikasi antara browser dan server dienkripsi menggunakan TLS
  • Autentikasi berbasis email: Akses hanya diberikan kepada email yang terdaftar oleh Administrator
  • Kontrol akses peran: Setiap pengguna hanya dapat mengakses data sesuai perannya
  • Token OAuth lokal: Token Google Drive tidak disimpan di database kami, hanya di browser pengguna
  • Database terenkripsi: Data disimpan di Neon PostgreSQL dengan enkripsi at-rest
Meskipun kami menerapkan langkah keamanan terbaik, tidak ada sistem yang 100% aman. Segera laporkan kepada Administrator jika Anda mencurigai adanya akses tidak sah.
5

Penyimpanan & Retensi Data

Data disimpan selama akun pengguna aktif dalam sistem. Ketentuan retensi:

  • Data usulan dan verifikasi disimpan permanen untuk keperluan audit dan pelaporan dinas
  • Akun pengguna yang dinonaktifkan tidak dihapus, hanya diblokir aksesnya
  • File data dukung di Google Drive sepenuhnya berada di bawah kendali pemilik akun Google
  • Penghapusan data hanya dapat dilakukan oleh Administrator sistem
6

Hak Pengguna

Sebagai pengguna sistem VISPM, Anda berhak untuk:

  • Mengakses data: Melihat semua data yang tersimpan terkait akun Anda
  • Meminta koreksi: Meminta Administrator memperbaiki data yang tidak akurat
  • Mencabut akses Google: Kapan saja melalui Google Account Permissions
  • Pertanyaan privasi: Mengajukan pertanyaan terkait penggunaan data kepada Administrator
7

Perubahan Kebijakan

Kebijakan privasi ini dapat diperbarui sewaktu-waktu. Perubahan signifikan akan diberitahukan kepada pengguna melalui notifikasi dalam aplikasi. Tanggal pembaruan terakhir selalu tercantum di bagian atas halaman ini.

Dengan terus menggunakan aplikasi VISPM setelah perubahan kebijakan, Anda dianggap menyetujui kebijakan yang diperbarui.

Ada Pertanyaan tentang Privasi?

Hubungi Administrator sistem VISPM di instansi Anda untuk pertanyaan terkait data dan privasi.
